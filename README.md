# ASMODAX-OPS-amodax-writeups
Write-ups técnicos, laboratorios y documentación ofensiva desarrollados bajo el protocolo ASMODAX. Cada entrada representa una operación simbólica: infiltración, explotación, persistencia y análisis. Este repositorio compila retos de plataformas como TryHackMe, HackTheBox y eJPT docker labs, con enfoque narrativo, táctico y técnico.
